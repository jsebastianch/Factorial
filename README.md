# Factorial

Este repositorio contiene una implementación en Java para calcular el factorial de un número. El proyecto está diseñado para ser simple y fácil de entender, ideal para quienes están aprendiendo conceptos básicos de programación en Java.

## Características

- Cálculo del factorial de números enteros.
- Código limpio y comentado.
- Ejemplo de uso incluido.

## Requisitos

- Java 8 o superior

## Uso

1. Clona el repositorio:
    ```bash
    git clone https://github.com/tu-usuario/Factorial.git
    ```
2. Compila el proyecto:
    ```bash
    javac Factorial.java
    ```
3. Ejecuta el programa:
    ```bash
    java Factorial
    ```
